# 🎨 UI Design Templates Collection

A curated collection of **27 standalone HTML/CSS UI design templates**, each showcasing a distinct visual style and design aesthetic — from futuristic cyberpunk to soft pastels and everything in between.

## 🚀 Getting Started

No build tools or dependencies required. Simply open any `.html` file directly in your browser, or launch the showcase index:

```
index.html  ← Start here for the full gallery
```

## 🛠️ Tech Stack

- **HTML5** — Semantic structure
- **CSS3** — Animations, gradients, custom properties, and modern layout
- **Vanilla JavaScript** — Lightweight interactivity where needed

## 📁 Project Structure

```
UI-design/
├── school-landing.html
├── school-dashboard.html
├── ...                    
└── README.md
```

## 💡 Usage

Each template is **fully self-contained** in a single `.html` file — copy any file to kickstart a new project or use it as a design reference.
